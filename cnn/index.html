CNN 전체 구조

Convolutional Neural Network의 전체 파이프라인입니다. 각 블록을 클릭하면 해당 섹션으로 이동합니다.

💡 왜 CNN인가?

완전연결(FC) 네트워크의 한계: 이미지를 1차원으로 펼치면 공간 구조가 사라지고, 파라미터 수가 폭발합니다 (28×28 이미지만 해도 784개 입력).

CNN의 해결: 커널을 공유하여 파라미터를 극적으로 줄이고(3×3=9개), 지역 패턴(엣지, 텍스처)을 계층적으로 학습합니다.

합성곱 연산 (Convolution)

💡 왜 합성곱인가?

가중치 공유: 하나의 커널이 입력 전체를 슬라이딩하므로, 같은 패턴을 위치에 관계없이 감지합니다.

지역 연결: 각 출력 뉴런이 입력의 일부 영역만 보므로, 엣지·코너 같은 지역 특징을 효율적으로 추출합니다.

계층적 학습: 얕은 층은 엣지, 깊은 층은 물체 부분 → 전체 객체를 단계적으로 인식합니다.

프리셋 커널:
학습 커널 선택: 실제 학습된 모델의 값
🐍 PyTorch 코드 
import torch.nn as nn

conv = nn.Conv2d(
    in_channels=1, out_channels=6,
    kernel_size=3, stride=1, padding=0
)
# 입력: [1, 1, 7, 7] → 출력: [1, 6, 5, 5]

다채널 합성곱

RGB 3채널 입력에 여러 필터를 적용하여 다수의 출력 채널을 생성합니다.

💡 왜 다채널인가?

다양한 특징 동시 추출: 채널마다 다른 패턴(수평 엣지, 수직 엣지, 텍스처 등)을 학습합니다.

정보 보존: 단일 필터로는 색상·밝기·질감 정보를 동시에 포착할 수 없으므로, 여러 필터가 상호 보완적으로 동작합니다.

🐍 PyTorch 코드 
conv = nn.Conv2d(
    in_channels=3, out_channels=1,
    kernel_size=3, stride=1, padding=0
)
# 입력: [1, 3, 5, 5] → 출력: [1, 1, 3, 3]

텐서 차원 변화

각 레이어를 통과할 때 텐서 shape이 어떻게 변화하는지 추적합니다.

💡 왜 차원 추적이 중요한가?

Shape Mismatch 디버깅: 딥러닝에서 가장 흔한 에러가 텐서 크기 불일치입니다. 각 레이어의 입출력 shape을 미리 계산하면 이를 예방합니다.

메모리 계획: shape을 알면 GPU 메모리 사용량을 예측할 수 있어, 배치 크기 결정에 도움됩니다.

프리셋:

풀링 (Pooling)

공간 차원을 줄여 계산량을 감소시키고 주요 특징을 추출합니다.

💡 왜 풀링인가?

계산량 감소: 특성맵 크기를 절반으로 줄이면 다음 레이어의 연산량이 1/4로 줄어듭니다.

평행이동 불변성: 물체가 이미지 내에서 약간 이동해도 풀링 결과가 유사하므로, 모델의 위치 강건성이 향상됩니다.

과적합 방지: 공간 해상도를 줄여 파라미터 수를 제한합니다.

🐍 PyTorch 코드 
pool = nn.MaxPool2d(kernel_size=2, stride=2)
# 입력: [1, 6, 26, 26] → 출력: [1, 6, 13, 13]

활성화 함수 (Activation)

비선형 활성화 함수가 값을 어떻게 변환하는지 확인합니다.

💡 왜 비선형 함수인가?

선형의 한계: 선형 변환을 아무리 깊게 쌓아도 결국 하나의 선형 변환과 같습니다. XOR 문제조차 풀 수 없습니다.

범용 근사기: 비선형 활성화를 추가하면 이론적으로 모든 연속 함수를 근사할 수 있습니다 (Universal Approximation Theorem).

ReLU의 장점: 양수에서 기울기가 1이므로 그래디언트 소실 문제를 완화합니다.

Flatten & Fully Connected

피처맵을 1차원 벡터로 펼친 뒤 FC 레이어를 통해 분류합니다.

💡 왜 Flatten & FC인가?

공간 → 의미: 합성곱으로 추출한 공간 특징을 하나의 벡터로 합쳐, 클래스 확률로 변환하는 다리 역할입니다.

비선형 결합: FC 레이어는 모든 특징을 전역적으로 조합하여, "이 조합이 고양이에 가깝다"는 판단을 내립니다.

Softmax: 최종 출력을 확률 분포로 변환하여, 가장 높은 확률의 클래스를 예측으로 선택합니다.

🐍 PyTorch 코드 
flatten = nn.Flatten()          # [1, 4, 3, 3] → [1, 36]
fc = nn.Linear(36, 10)          # 36차원 → 10 클래스
output = torch.softmax(fc(flatten(x)), dim=1)
